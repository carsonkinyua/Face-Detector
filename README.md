# Face Detector Sample App

This is a sample Android application demonstrating the implementation of face detection using **Google ML Kit**.

## Features
- Real-time or static image face detection.
- Modern UI built with **Jetpack Compose**.
- Dependency injection using **Koin**.
- Efficient image loading with **Coil**.

## Tech Stack
- **Kotlin** - Primary programming language.
- **Jetpack Compose** - Modern toolkit for building native UI.
- **ML Kit Face Detection** - Google's machine learning SDK for detecting faces.
- **Koin** - Pragmatic lightweight dependency injection framework.
- **Coil** - Image loading library for Android backed by Kotlin Coroutines.

## Getting Started

### Prerequisites
- Android Studio Ladybug or newer.
- Android SDK 24 or higher.

### Installation
1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle.
4. Run the app on an emulator or a physical device.
